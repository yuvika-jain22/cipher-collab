# Cipher Colab Frontend

React 19 + Vite client for the Cipher Colab collaborative developer workspace.

## Setup

```bash
npm install
cp .env.example .env
npm run dev
```

## Environment

- `VITE_API_BASE_URL`: REST API base URL.
- `VITE_WS_BASE_URL`: WebSocket base URL.

## Scripts

- `npm run dev`: start the local Vite server.
- `npm run build`: create the production build.
- `npm test`: run Vitest.
